# 國教盟知識王 🎯

第四屆會員大會互動問答遊戲

**線上版：** https://weall888-jpg.github.io/naer-quiz/

---

## 專案概述

- **用途：** 會員大會現場互動，測驗會員對國教盟三年倡議成果的了解
- **題目數：** 題庫 30 題，每次隨機抽 15 題
- **時間限制：** 每題 15 秒
- **特色：** 語音播報、吐槽評語、連擊特效、結尾分級評價

---

## 技術架構

### 單檔 HTML（零依賴）
- 所有程式碼在一個 `index.html` 內
- 不需要伺服器，直接開啟就能用
- 適合 GitHub Pages 靜態部署

### 核心技術
```
HTML5 + CSS3 + Vanilla JavaScript
├── Web Audio API（音效合成）
├── Web Speech API（語音播報）
├── CSS Animations（動畫特效）
├── Canvas（結束彩帶）
└── LocalStorage（可擴充存分數）
```

---

## 功能模組

### 1. 音效系統
```javascript
// Web Audio API 合成音效，不需要音檔
playSound('correct')  // 答對音效
playSound('wrong')    // 答錯音效
playSound('tick')     // 倒數滴答
playSound('win')      // 勝利音效
```

### 2. 語音播報
```javascript
// 使用瀏覽器內建 TTS
speak("答對了！太厲害了！", 1.1, 1.05)
// 參數：文字、語速、音調
```

### 3. 吐槽系統
- `correctRoasts[]` - 20 句答對吐槽
- `wrongRoasts[]` - 20 句答錯吐槽
- `timeoutRoasts[]` - 8 句超時吐槽
- `comboRoasts[]` - 連擊專屬吐槽（3連~10連）
- 不重複機制：用過的會標記，全用完才重置

### 4. 結尾評價（依答對數分級）
```
15題：國教盟之神 👑
14題：秘書長等級 🥇
12-13題：資深會員認證
10-11題：常關注齁？
8-9題：中上程度
6-7題：勉強及格 😅
4-5題：加入那天在睡覺？😴
2-3題：用猜的對吧？🎲
1題：機率比亂猜還低 🙈
0題：走錯棚了？🚪
```

### 5. 視覺特效
- 粒子背景動畫
- 答對/答錯閃光
- 答錯螢幕震動
- 連擊星星爆炸
- 結束彩帶慶祝

### 6. 主持人控制（電腦版）
- `空白鍵`：直接揭曉答案（給分）
- `→`：下一題
- `R`：重新開始

## 題庫結構

```javascript
{
  category: "📊 數據題",           // 題目分類標籤
  question: "問題文字...",         // 題目
  options: ["A", "B", "C", "D"],   // 四個選項
  correct: 1,                      // 正確答案索引 (0-3)
  detail: "補充說明..."            // 揭曉後顯示的說明
}
```

### 題目分類
- 📊 數據題（新聞稿數量、媒體聲量等）
- 🏆 排名題（最多報導、最多議題等）
- 📅 時間題（事件發生年份）
- 🤝 合作題（立委、團體、專家數量）
- ⚠️ 警訊題（倍數增長數據）
- 📈 趨勢題（變化趨勢）
- 🎯 議題題（議題分類）
- 🏛️ 政策題（政策倡議）
- 👥 青年部（青年部調查）
- 📰 新聞題（特定新聞報導數）
- 🎓 活動題（活動場次）
- 🔥 綜合題

## 開發流程

### 1. 規劃

- 語音播報
- 動畫特效

- 本機測試
- 部署到 GitHub Pages
